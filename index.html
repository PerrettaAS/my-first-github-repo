A* Search Algorithm Research Project

While taking Data Structures at Central Connecticut State University, I was given an assignment to research a graph algorithm of my choice. The task was to chose, implement, and present the graph algorithm. After some research, the A* graph search algorithm caught my eye, and I decided to complete my project using it.

The A* search algorithm is a graph search algorithm that determines an efficient traversal path from one node to another. It combines elements of Dijkstra's algorithm and greedy best-first-search to efficiently determine the shortest path. For my implementation, I created a 2D tile-map game where a character can walk on any grassy terrain, while prohibited to walk on any rocky terrain. The implementation of the algorithm allowed the player to click on any grassy tile, and take the shortest path to get there. I chose this type of implementation because this algorithm is popular with different tile-map based games. You can find this project on my github here.

Image of the tile-based game that I created to demonstrate the functionality of the A* search algorithm.

Calendar+

Calendar+ is a time-management meteor based web application that allows users to quickly and easily create and manage their calendar. The inspiration behind designing this application came from the frustration of creating my own schedule. When I use different time-management applications, most have many features I don't need that just waste my time when creating my calendar. I wanted to design an application that has only the essential features needed to maximize my efficiency when adding events and tasks to my calendar.

The core features that I implemented into this web application are adding and removing events to an infinite calendar. I used the FullCalendar lib to create the infinite calendar and used their event system to help generate the events. Each event is customizable to the users liking and creates a flexibile experience. In the future, I plan on implementing shared calendars, allowing users to add events to a group calendar. You can find this project on my github here.
