A* Search Algorithm Research Project

While taking Data Structures at Central Connecticut State University, I was given an assignment to research a graphing algorithm of my choice. The task was to research, implement, and present the algorithm we chose. After some research, the A* search algorithm caught my eye, and I decided to complete my project using this algorithm.

The A* search algorithm is a graphing algorithm that determines an efficient traversal path from one node to another. It combines elements of Dijkstra's algorithm and greedy best-first-search to efficiently determine the shortest path. For my implementation, I created a 2D tile-map game where a character can walk on any grassy terrain, while prohibited to walk on any rocky terrain. The implementation of the algorithm allowed the player to click on any grassy tile, and take the shortest path to get there. I chose this type of implementation because this algorithm is popular with different tile-map based games.

NoteShare

NoteShare is my second individual project I developed in the A100 program. This application is a tool where you can create notes and post them online and share them to everyone on the site. You will also have the ability to create private notes for yourself and groups of people that you'd like to share specific notes with. For now, the notes are all publicized, however, this is a work in progress and I will continue to work on this project over time.

There are several features I plan on implementing into this project. One feature I'd like to have is group notes. Each account can create different groups of friends, and they can then share notes within this group. This will allow private notes that you can share with anyone you want to. Another feature I'd like to have his public notes within an area. Essentially, there would be a section of notes that were created and publicized near the user's location. These are some of the future features you'll see in this project as I continue to work on it.
